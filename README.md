# 🏠 AI House Price Predictor

This project predicts house prices using machine learning.

## 🚀 Features
- Predict house prices based on input features
- Simple web interface
- Built with Python & ML

## 🧠 Tech Stack
- Python
- Scikit-learn
- Pandas
- Flask

## 📊 How it Works
1. Train model on housing dataset
2. Save model using joblib
3. Load model in web app
4. User inputs → prediction output
